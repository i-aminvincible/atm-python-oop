# ATM System (Python OOP Project)

A command-line ATM simulation built using Python, demonstrating core Object-Oriented Programming concepts.

## Features
- Create/set a secure PIN
- Deposit money
- Withdraw money (with insufficient balance check)
- Check account balance
- PIN verification before every transaction

## Concepts Used
- Classes and Objects
- Encapsulation (private attributes using `__pin`, `__balance`)
- Control flow with `while` loop for continuous menu
- Conditional statements (if/elif)

## How to Run
```bash
python Class.py
```
